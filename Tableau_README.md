# 📊 Sales Dashboard Project (Tableau)

## 📌 Description
This project presents an interactive sales dashboard built using Tableau. 
It helps analyze business performance using key metrics and visualizations.

---

## 🎯 Objectives
- Track overall sales performance  
- Analyze category-wise and region-wise trends  
- Identify key business insights  

---

## 📊 Key Features (KPIs)
- Total Sales  
- Total Profit  
- Total Quantity  
- Number of Orders  

---

## 📈 Visualizations Used
- Bar Chart → Sales by Category  
- Bar Chart → Sales by Region / City  
- Donut Chart → Category Contribution  
- KPI Cards → Summary metrics  

---

## 📂 File Information
- `.twbx` → Tableau Packaged Workbook (includes dashboard + dataset)

---

## 🚀 How to Use
1. Download the `.twbx` file  
2. Open in Tableau Desktop or Tableau Public  
3. Interact with filters and visuals  

---

## 📊 Insights
- Identify top-performing categories  
- Compare sales across regions  
- Understand contribution of each category  
- Track business growth trends  

---
